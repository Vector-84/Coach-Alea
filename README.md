# Coach Alea

Application web de fitness installable sur smartphone sous forme de PWA.

## Publication simple

Le projet est statique, donc tu peux le publier tel quel sur GitHub Pages ou Netlify.

### Option 1: GitHub Pages

1. Mets le projet dans un dépôt GitHub.
2. Va dans les paramètres du dépôt.
3. Ouvre la section Pages.
4. Choisis la branche `main` et le dossier racine `/`.
5. Sauvegarde puis attends l’URL de publication.

### Option 2: Netlify

1. Connecte ton dépôt à Netlify, ou glisse-dépose le dossier du projet.
2. Garde la configuration par défaut.
3. Publie le site.

## Installation sur smartphone

### Android

Ouvre le site dans Chrome ou Edge, puis utilise l’option d’installation ou “Ajouter à l’écran d’accueil”.

### iPhone

Ouvre le site dans Safari, puis “Partager” et “Sur l’écran d’accueil”.

## Notes

- Le site doit être servi en HTTPS pour que le service worker fonctionne.
- Les icônes PWA sont dans `icon-192.png` et `icon-512.png`.
